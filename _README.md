This is a PHP comments extension write in DDD style.

## Install

Via Composer

```bash
$ composer require vova07/ddd-comments
```

## Usage

```php
$skeleton = new vova07\Skeleton();
echo $skeleton->echoPhrase('Hello, Yii2!');
```

## Testing

```bash
$ phpunit
```
